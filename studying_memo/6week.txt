git bash 명령어

ls -> 현재파일에 어떤 리스트들이 있는 뽑아줌
pwd -> 현재 디렉토리의 위치를 알려줌(현재폴더위치)
cd -> 폴더이동
cd .. -> 직전 폴더로 이동
mkdir -> 폴더 생성
rm -rf 삭제할 폴더명 -> 폴더 삭제
touch -> 파일생성
rm -f 삭제할 파일명 -> 파일 삭제
cp (복사할 파일)(복사한파일) -> 파일 복사
mv (이동할 파일)(이동시킨 후 파일 위치) -> 파일 이동


[절대 하면 안되는 명령어~!!!! 이 명령어를 이용하면 포맷할수있다. 복구불가능]
rm -rf * -> 모든거 싸그리 삭제~
